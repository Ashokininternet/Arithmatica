# Arithmatica 🧮

Arithmatica is a modern, lightweight web application designed to make practicing mental math simple and engaging. Whether you're a student sharpening your skills or an adult looking to keep your mind quick, Arithmatica provides a streamlined platform for arithmetic mastery.

## 🚀 Live Demo
Check out the app in action here: 
**[https://ashokininternet.github.io/Arithmatica/](https://ashokininternet.github.io/Arithmatica/)**

## ✨ Features
- **Core Function:** Does Calculations related to physics and maths. eg: Distance, Time, and Acceleration and many more.
- **Instant Feedback:** Get real-time results on your answers to learn from mistakes immediately.
- **Responsive Design:** Works seamlessly on desktops, tablets, and mobile phones.
- **Minimalist Interface:** No clutter, just math.

## 🛠️ Built With
- **HTML5** - Semantic structure.
- **CSS3** - Responsive and clean styling.
- **JavaScript (ES6+)** - Logic for problem generation and score validation.
